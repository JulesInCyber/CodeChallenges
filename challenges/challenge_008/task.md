# Challenge 008: Number Pyramid

**Difficulty**: Easy

## Task

Write a function that takes a number as an input.
The function should return `n` lines of the number where each line has one more repetition of the number.

## Example

```bash
pyramid(1)
>> 1

pyramid(2)
>> 2
>> 22

pyramid(5)
>> 5
>> 55
>> 555
>> 5555
>> 55555
```
