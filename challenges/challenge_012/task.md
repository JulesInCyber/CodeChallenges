# Challenge 012: Friday the 13th

**Difficulty**: Medium

## Task

Write a function that takes two inputs:

- a month (1-12)
- a year

The function should return `True`if the 13th of the given month and year is a friday.
If not, the function shoult return `False`

## Example

```bash
friday_13th(8,2021)
>> True

friday_13th(11,2019)
>> False
```
